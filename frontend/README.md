# Frontend

React + Vite + Tailwind. Dark-theme portfolio and chatbot UI.

## Run

```bash
npm install
npm run dev
```

App runs on http://localhost:5000 (host `0.0.0.0`). API calls use relative URLs `/api/health` and `/api/ask`; Vite proxies them to the backend (port 8000). Start the backend so the chat and health indicator work.
